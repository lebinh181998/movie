- Project: Movie

- Introduction: A website used to view poster, trailer of movies
- Functional:
	+ View movie detail: When click poster, a component detal for movie will show at the bottom of website. You can watch trailer or teaser if it available.
	+ Search movie: Click Search icon at the top right to go to Search page. You can enter movie name and choose attributes.	
- Link demo: https://lab-20-1-f4f76.web.app
- deployment guide: Change the url to http://localhost:5000 in a few file and use-input file in hook folder.
